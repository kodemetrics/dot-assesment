# dot-assesment command to run(Spring boot):


## Step 1
```
git clone https://github.com/kodemetrics/dot-assesment

cd dot-assesment
```

## Compiles to jar
```
mvn clean package
```

### Run command on terminal or cmd
```
java -jar target/DOT-0.0.1-SNAPSHOT.jar --accessFile=C:\Users\kingsley.okafor\Desktop\user_access.txt  --start=2022-01-01.13:00:00 --duration=daily --limit=250
```
